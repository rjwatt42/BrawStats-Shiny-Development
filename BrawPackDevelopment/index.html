Go to the top of the Jamovi window.
There is a row of menu options: Variables, Data, Analyses, Plots, Edit
Beneath each is a set of icons: Exploration, T-Tests, ANOVA etc

Analysis: choose the menu Analyses
  1. Press the Regression icon
    & choose Correlation Matrix from the drop down menu
  2. Now
    • DV to Variables
    • IV to Variables
  3. Set the option
    • Pearson in Correlation Coefficients
Graph: choose the menu Plots
  1. Press the Scatter Plot icon
  2. Now move
    • DV to Y-Axis
    • IV to X-Axis
  3. Set the option
    • Linear in Regression Line
Effect Size: choose the menu Analyses
  1. Press the Regression icon
    & choose Linear Regression from the drop down menu
  2. Now move
    • DV to Dependent Variable
    • IV to Covariates
  3. Select Model Fit options group and set
    • R in Fit Measures
